== Description ==
RDS Smarty is a fast, flexible, Astra-inspired theme with a bold tech and commerce-ready aesthetic. Built for WordPress 6.4+ and PHP 7.4.

== Installation ==
1. Upload the theme to the /wp-content/themes/ directory.
2. Activate the theme through the Appearance menu.
